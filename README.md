# Power_BI_Dashboard_Super_Store
# 📊 Super Store Sales Dashboard

## Overview
A professional Power BI dashboard built to analyze Super Store sales performance, profitability, customer behavior, and regional trends. This project transforms raw business data into actionable insights through interactive visualizations and KPI tracking.

![image alt](https://github.com/harshitdara66-sys/Power_BI_Dashboard_Super_Store/blob/2ea8d15727b5c00d6ef41b0db2eda4568cf9bef3/Screenshot%202026-06-23%20112837.png)
## Dashboard Highlights
- Revenue & Profit Analysis
- Sales Performance Monitoring
- Regional & State-wise Insights
- Category and Sub-Category Analysis
- Customer Segment Performance
- Trend Analysis Over Time
- Interactive Filters and Drill-through Features

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Order Count
- Average Order Value
- Regional Performance Metrics

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Business Intelligence & Analytics

## Business Insights
The dashboard helps stakeholders:
- Identify top-performing products and categories
- Monitor profitability across regions
- Analyze customer purchasing behavior
- Track sales trends and growth opportunities
- Support data-driven decision making

## Getting Started

### Prerequisites
- Microsoft Power BI Desktop

### Open the Dashboard
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Refresh the data if required.
4. Explore the interactive reports and visualizations.
